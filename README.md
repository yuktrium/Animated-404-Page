# 🌩️ Animated 404 Error Page

A premium and interactive **Animated 404 Error Page** built using **HTML, CSS, and JavaScript**. This project features a cute animated mascot, rain effects, lightning flashes, emotional character animations, and a modern UI to transform a standard 404 page into an engaging user experience.

## 🚀 Live Demo

https://yuktrium.github.io/Animated-404-Error-Page/

## ✨ Features

* 🌩️ Animated rainy background
* ⚡ Lightning flash effects
* ☁️ Smooth moving clouds
* 💛 Cute animated mascot
* 🥺 Emotional story-based animation
* 💧 Tear animation
* ✨ Interactive hover effects
* 🎭 Smooth timeline animations
* 📱 Fully responsive design
* ⚡ Built with pure HTML, CSS & JavaScript
* 🔧 Easy to customize

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
